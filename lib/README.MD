# Arquivos de script

Esse diretório contém os diversos scrips desenvolvidos pela Rech para ser usado pelo Atom.

## busca.js

Arquivo que contém scripts de busca, como por exemplo
* Busca de declaração (F12)

## comentarios.js

Scripts para digitação e manipulação de comentários, como:
* Insere comentários (Alt+N)
* Insere separador (Alt+-)

## clipboard.js

Operações para manipular o clipboard
* Copiar/colar linha (F10, F11)

## open.js

Arquivo com scripts para abertura de fontes ou diretórios
* Abre fontes de consulta (Ctrl+Shift+O)
* Abre fontes no WC (Alt+O)

## rech.js

Esse é o arquivo principal do pacote Rech. Ele contém os métodos de ativação do pacote Atom, que faz com que ele seja incluído ao Atom.

## commons

Contém diversas subrotinas, como manipulação de texto e arquivos

## commons-view

Contém views (Componentes visuais) de uso commum a diversas funcionalidades

## editor

Scripts para comunicação com o editor (Encapsulam rotinas do editor Atom)

## identa

Módulo que contem as rotinas de identação

## linter-cobol

Scripts para fazer o *linter* (Hints com warnings e erros) dos fontes COBOL
