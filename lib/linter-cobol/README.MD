# cobol-linter

Esse pacote provê suporte a *linter* para fontes COBOL. *Linter* são os hintes, erros, warnings ou informações que são exibidos durante o desenvolvimento de um fonte para garantir qualidade de código fonte e reduzir a chance de bugs ou erros.

O *linter* COBOL utiliza diversas ferramentas para criar os warnings e erros, que estão listadas abaixo.

## Pré-processador

O pré-processador gera diversos warnings que indicam melhorias ou possíveis bugs no código fonte.
