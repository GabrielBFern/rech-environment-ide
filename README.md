# Pacote Rech

Este pacote engloba os diversos scripts, funções e ferramentas desenvolvidas pela Rech para uso no nosso ambiente de desenvolvimento.

## Estrutura do pacote

### Diretório "keymaps"

Contém a definição das teclas de atalho para as diversas funções do pacote

### Diretório "lib"
O diretório lib contém a maior parte dos nossos Scripts

### Diretório "menus"

Contém a definição dos menus que serão criados no Atom

### Diretório "snippets"

São os *snippets*, templates de auto completamento, que são preenchidos com o *tab*

### Diretório "styles"

Contém arquivos CSS para estilização dos componentes do Atom e componentes nossos

### Arquivo "package.json"

Esse arquivo contém as definições do pacote, como o nome, versão e dependências
